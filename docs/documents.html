Introduction

Open color is a color sheme for UI design. You can use it for font, background, border etc. It contains a gray and 12 colors. All the color is perceptible by color blind (specifically deuteranopia and protanopia) person.

Installation

{% highlight bash %}$ npm install open-color{% endhighlight %}

Variable convention

{% highlight scss %}$oc-(color)-(number){% endhighlight %}
  • oc Abbreviation for Open color
  • (color) Color name like gray, red, lime ect.
  • (number) 0 to 9. Color brightness spectrum

How to use

Import the file to your project and use the variables.

Example

{% highlight scss %}//SASS, SCSS @import 'path/open-color'; .body { backgrond-color: $oc-gray-0; color: $oc-gray-7; } a { color: $oc-teal-7; &:hover, &:focus, &:active { color: $oc-indigo-7; } } {% endhighlight %}
